Propriedade length

Método IndexOf

Este exemplo verifica se uma string (variável de texto) contém uma determinada palavra. Se a palavra for encontrada será indicada a posição em que foi localizado o seu primeiro caractere.
Nota: A posição do primeiro caractere da string é 0, não é 1.

Método Match

Este exemplo verifica se uma palavra (que neste caso é “gelada”) está contida na string. Se estiver ela será devolvida.
